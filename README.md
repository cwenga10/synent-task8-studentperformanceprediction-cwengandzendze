# Student Performance Prediction

## Objective
Predict student overall performance using machine learning techniques.

## Dataset
Student Performance Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Visualization

## Model
Random Forest Regressor

## Results
MAE: 2.42
RMSE: 3.00
R² Score: 0.976

## Key Findings
- Student performance can be accurately predicted.
- Attendance and study habits are important factors.
- The model achieved excellent predictive accuracy.
